
这是我的博客[Aaron's Blog](https://aaroon.me/blog/)的源码仓库。博客使用Astro搭建，样式风格从[AstroPaper](https://github.com/satnaing/astro-paper)发展而来。

## 技术栈

- Astro
- React
- TypeScript
- TailwindCSS
- Markdown

## 特点

- PWA
- Lighthouse满分
- 响应式布局
- 无障碍
- SEO友好
- 浅色/深色模式
- 模糊搜索
- RSS订阅
